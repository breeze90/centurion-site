# Сайт Центурион

## Задача
Сделать новый сайт вместо текущего: https://centurion-group.ru/

Ориентир по объёму: ~10 страниц (+-).

## Структура папок

Весь сайт лежит в папке `site/` (корень для деплоя на хостинг).
Проектные решения по структуре и дизайну страниц фиксируются в `docs/`.

```
docs/
└── homepage-blocks.md  ← карта блоков главной и принятые решения

site/
├── index.html      ← Главная (пока единственная страница)
└── assets/
    ├── css/
    ├── js/
    ├── fonts/
    └── img/
```

Остальные разделы (о компании, услуги и т.д. — по меню centurion-group.ru) будем добавлять папками по мере расширения под SEO.

## Статус
Главная страница опубликована как рабочий черновик. Актуальная структура блоков, статус текстов и открытые вопросы описаны в `docs/homepage-blocks.md`.
